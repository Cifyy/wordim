## Usage 

Enter a pattern in `pattern.txt` and run the program.
Example:
  - 22222
  - 10101
  - 01010
  - XXXXX
  - 00010
  - 00102

Make sure `valid-wordle-words.txt` is in the same directory.
Run the program.

Symbols:
   - `0` = gray 
   - `1` = yellow
   - `2` = green
   - `X` = either green or yellow
